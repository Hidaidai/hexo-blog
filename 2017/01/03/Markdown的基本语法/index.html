Markdown的基本语法

Jan 3, 2017 | Web设计 | Hits

  对于基于Hexo制作的个人博客，因为没有后台，对于文档的编写都是在本地进行的，而作为一个网页，需要对文本内容进行排版以及插入表格图片等，这些都是通过Markdown实现的。其实刚开始使用Markdown是很不习惯的，但在写完前面几篇博文后，对与Markdown也开始熟悉起来，确实简介明了，功能强大，虽然不能可见即可得，但通过Markdown的编辑器是可以实现的。初学者对Markdown的语法还是有点难的，在学会基本语法之后还是需要多练习的。

一、了解Markdown

  Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown应用于文档快速排版，简单易读，编写过程双手基本不需要离开键盘，就可以写出格式丰富而又美观的文章。Markdown的文件后缀名为.md，在网页上呈现还是需要解析成HTML，其支持的HTML标记仅仅是一部分，所以Markdown可以看成是HTML一个简化了标记形式的子集。不过它有着它自己的一系列特点：

  • 语法简单，仅仅需要一些符号，就完成排版
  • 快速编写，不需要了解HTML的标签，不需要用到Word的各种按钮
  • 直接阅读，没有HTNL标签的干扰
  • 渲染效果，网页显示效果简洁明了，和HTML设计的网页不相上下

  Markdown具有一系列衍生版本，用于扩展Markdown的功能（如表格、脚注、内嵌HTML等等），这些功能原初的Markdown尚不具备，它们能让Markdown转换成更多的格式，例如LaTeX，Docbook。

3.排版——————–

  如果要回车，另起一行需要在段落最后输入两个空格。否则一直都是一个段落。或者在段落开头输入&emsp;&emsp;对于&emsp;大家应该都比较熟悉，表示空格。我就是这样另起一行然后首行缩进。

4.表格——————–
